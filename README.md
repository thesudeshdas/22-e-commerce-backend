## Run Locally

1 - Clone the project

```bash
  git clone https://github.com/thesudeshdas/dashtra-backend.git
```

2 - Go to the project directory

```bash
  cd dashtra-backend
```

3 - Install dependencies

```bash
  npm install
```

4 - Add environment variables

```bash
  MONGO_DB_URI
```

5 - Start the server

```bash
  npm run serverstart
```
